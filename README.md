# 4-bit Adder using Verilog

## Description
This project implements a 4-bit adder using Verilog HDL.

## Features
- Adds two 4-bit inputs
- Generates sum and carry
-easy to extend to higher bit adder

## Tools Used
- Xilinx Vivado
- Verilog HDL

## How to Run
1. Open Vivado
2. Add source code
3. Run simulation

## Output
Correct addition of binary inputs
